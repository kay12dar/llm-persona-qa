# LLM Persona QA

Testing role consistency, gating, and memory behavior of LLM-powered chat personas.

## What's tested
- Role stability under direct questioning
- System prompt confidentiality
- Gating on sensitive topics
- Memory retention within a session
- Personality consistency

## Files
- `system_prompt.txt` — test persona used for all scenarios
- `bug_reports.md` — documented behavior failures with reproduction steps

## Tools used
- Claude API / ChatGPT
- Manual testing (exploratory + scripted scenarios)
- Markdown for documentation
